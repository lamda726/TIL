R 개요

1. R 라이브러리 중 java 기반으로 되어 있는 놈들을 사용할 수 있도록 JDK를 먼저 설치

2. R은 통계 계산을 위한 프로그래밍 언어이면서 데이터 분석 소프트 웨어.
- AT&T의 Bell연구소 S라는 통계언어에서 문법부분 통계처리 부분을 참고해서 만들어 냄.
- 1993년 통계학과 교수 로스 이하카와 로버트젠틀맨 2명이서 개발
  --> 일반 사람들도 쉽게 통계적 데이터 분석을 할 수 있도록 개발

3. R로 무엇을 할 수 있을까?
- 기본적 통계 분석: 기초 통계, 가설 검정
- 머신러닝 모델링: SVM, Random Forest, Deep Learning
- 텍스트 마이닝: 문장 속의 단어 빈도를 이용하여 글의 성향을 분석
- 이미지 분석, 사운드 분석

4. 데이터 분석을 할 수 있는 다른 패키지는 무엇이 있을까?
- SAS, SPSS: 기능은 R보다 더 좋지만, 유료!! 최적화가 되어있는 프로그램
- Excel: 무료이면서 통계 처리 가능. But 기능이 너무 빈약, 셀이 13만 line을 넘어가면 처리 불가능
- R: 기능은 유료에 버금가면서 무료!!

5. 데이터 분석을 할 수 있는 다른 프로그래밍 언어는 어떤게 있을까?
- R: 일반적으로 데이터 분석 후 결과를 어딘가에 Reporting을 하거나 차트로 표현하는데 최적 (분석 자체에 목적이 있다)
- Python: 데이터 분석 후, 해당 결과를 다른 시스템에 연동시키는 목적으로 사용.

* R로 연동을 못하는 것은 아니다. 하지만 파이썬으로 해당 작업을 처리하는 것이 보다 쉽고 편리

6. R의 장점
 - 오픈 소스: 무료로 사용할 수 있어요!
 - 오픈 소스 생태계: CRAN 사이트에서 여러 패키지를 공유, 최신 분석 기법들을 손쉽게 이용 가능!
 - 프로그래밍 방식의 데이터 분석: 재현성을 확보할 수 있다!
   (프로그래밍 방식은 코드레벨에서 데이터 분석을 하기 때문에 가능한 것)

7. R 다운로드
- CRAN에서 platform에 맞는 파일을 다운로드 후 설치
 => windows 시스템에 설치할 때 로그인 아이디가 한글인 경우에는 설치가 안돼요!(영문 ID를 생성한 후 설치)
- IDE(개발툴)이 필요!!! : RStudio를 다운로드해서 설치


8. Rstudio 실행 : 화면구성을 알아보자!
 - Source 창: 스크립트 파일을 생성해서 코드를 작성해요(블록처리 통해 여러개의 실행 결과를 동시에 출력 가능)
 - Console 창: 코드의 실행결과가 출력돼요!
 - Environment 창(환경창): 코드 실행 시 사용된 변수의 값을 알려줌!
 - file 창: 현재 사용하는 폴더 ( working directory )를 확인할 수 있음
 - plot 창: 그래프, 도표 등
 - package 창: 사용하고 있는 패키지 종류 알려주는 창
